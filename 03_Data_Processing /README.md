

Обработка и анализ данных
==

#####Лекторы: Филипп Кац, Катя Серова
#####Даты: TBC

###Описание занятия:
	В рамках данного интенсива студенты знакомятся в разными типами информации о городе по ее источникам (государственная статистика (обзор статистики по Санкт­Петербургу); международные источники, ведомства; open data; cоциальные сети, сервисы, краудсорсинг: API, готовые решения, open source, apps, QuantifyYourself; железо: сенсоры, камеры, устройства, DIY, Hack, интерактивные инсталляции и проекты; данные бизнес-структур (сотовые операторы, торговые сети и др.)) и экспериментируют с обработкой открытых данных и информации из социальных сетей. 
	Студентам представляются ключевые требования к анализу для отдельных задач и групп конечных потребителей (транспортный анализ, градостроительный анализ, исследования для торговых сетей и рынка недвижимости), а также подходы к интерпретации пространственной информации о городе (распределение Ципфа, теория центральных мест В.Кристаллера, теория географического поля, пространственная структура городов по А.Берто и др.)
